# AtliQ Hospitality Dashboard Suite

## Business Problem
The Revenue team needed visibility into occupancy and pricing performance 
across properties to support dynamic pricing and occupancy optimization 
decisions.

## Approach
- Built two Power BI dashboards using star-schema data modeling
- Tracked key metrics: ADR, RevPAR, Occupancy %, Capacity
- Analyzed 3 months of hotel booking data across multiple properties
